# SDK for creating new strategies on Uniswap V3

## Getting started

```
git clone https://github.com/mellow-finance/mellow-strategy-sdk.git
cd mellow-strategy-sdk
python3 -m venv .venv
source .venv/bin/activate
pip install poetry==1.1.13
poetry install  
```
